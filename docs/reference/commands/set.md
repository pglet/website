Set control properties.

```bash
set <control-id>
  property1=value
  property2=value
  ...
```

## Examples

```bash
set greeting value="Welcome back, $USER!"
```

This command changes `value` property of text control with `greeting` ID to `Welcome back, $USER!`.

## Parameters

### control-id

Control ID to update properties.
